Setting up a personal velocity model database

  1. Decide on a directory to place your personal database into.  For the purpose of illustration assume this is $HOME/mymodels.
  2. Copy the default database tables from $ANTELOPE/data/tables/genloc/db to your personal database directory.
  3. Edit the mod1d table as described in  adding a new velocity model document.
  4. If desired you can delete one of the existing models in the default database, but this is normally not a very good idea as it can cause great confusion in communication with others.
